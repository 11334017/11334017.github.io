嗨你好，我依照Figma上的設計完成了這個網頁設計
並且多加了一點小功能

## 功能特色

- **互動式卡片設計**：滑鼠移到卡片上會有浮動效果和陰影
- **可點擊連結**：點擊球員圖片可跳轉至 NBA 官網的球員頁面(不過最後一位沒找到他在NBA官網的網頁，所以找了其他替代的網址，如有需要可以自行替換，或是在展示的時候就不要按)
- **響應式導航欄**：包含 HOME、ABOUT US、PLAYERS、NEWS及註冊登入皆具有 hover 動畫效果

### 色彩配置
- 主色調：橘紅色 (#ff5722)
- 背景：深色漸層 (#2c3e50 到 #000000)
- 卡片：白色背景 (#ffffff)

### 互動效果
- 導航連結 hover 時變色並顯示底線
- SIGN UP 按鈕 hover 時向上浮動並顯示陰影
- 球員卡片 hover 時向上移動 10px 並增強陰影
